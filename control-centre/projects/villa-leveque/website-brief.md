# Villa Leveque Website Brief

Project: Villa Leveque
Type: Premium Greek villa rental website
Location: Kassiopi, Corfu, Greece
Purpose: luxury private villa rental enquiries.

## Brand Feel
Luxury, calm, Mediterranean, elegant, premium, clean, spacious, trustworthy, private, high-end. Use warm white/ivory, deep charcoal, muted stone/beige/sand, subtle gold/bronze, and Mediterranean blue where appropriate.

## Required Pages
Home, The Villa, Gallery, Services, Location, Contact.

## Core CTA
Primary: Check Availability. Secondary: Enquire Now, Send Enquiry, Arrange This Service, View Gallery.

## Layout Rules
Centred containers, max-width 1200-1440px, desktop padding ~48px, tablet ~32px, mobile ~20px, desktop sections 80-120px, mobile 48-72px, grid gaps 24-40px. Image-led, responsive, elegant, not cramped.

## Components
SiteHeader, SiteFooter, SectionContainer, PageHero, CTAButton, AmenityCard, VillaFactCard, GalleryGrid, ServiceSection, ContactForm, ContactCard, FeatureCard, LocationCard. Structure: Container → Section → Grid → Card → Button → Typography → Image treatment.

## Page Requirements
### Home
Premium hero, large villa/Corfu imagery, Villa Leveque headline, concise luxury subheading, Check Availability + View Gallery, short intro, 3 feature cards, villa highlights, gallery preview, location teaser, final enquiry CTA.

### The Villa
Hero image, headline “Inside Villa Leveque”, centred intro, At a Glance icon cards, details layout, image-led sections, amenities grid, Enquire Now CTA. Include 3 En-suite Bedrooms, Private Pool, Outdoor Dining, Air Conditioning, High-Speed Wi-Fi, Fully Equipped Kitchen, Sea & Garden Views, Private Parking, Concierge Support. Icons: bedrooms, bathrooms, guests, pool, sea views, garden, Wi-Fi, air conditioning, parking, kitchen.

### Gallery
Large hero, editorial gallery, desktop 3 columns/tablet 2/mobile 1, consistent gaps/cropping.

### Services
Personal Concierge, Private Transfers, Private Chef, Pre-Arrival Groceries. Each has number, title, short premium description, image, CTA “Arrange This Service”, alternating layouts.

### Location
Hero image, Kassiopi/Corfu overview, two-column layout, travel times/highlights card, Things To Do grid, map placeholder, Check Availability CTA. Themes: beaches, harbour, restaurants, village atmosphere, airport transfer time, private stay near sea.

### Contact
Premium two-column enquiry page. Contact card: Email, Phone, WhatsApp, Location, response time. Form: Full name, Email address, Phone/WhatsApp, Arrival date, Departure date, Guests, Message. Submit: Send Enquiry. Helper: “No payment required. We’ll confirm availability and respond within 24 hours.”

No downloaded villa/travel/hotel/property/real-estate web templates were found in the current workspace. Existing Control Centre business pages provide reusable static-card/dashboard patterns only. Use premium hospitality layout patterns from the brief until real reference templates are added.
